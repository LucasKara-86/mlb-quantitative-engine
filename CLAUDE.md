# Instruções do projeto

- Sempre que houver opções a serem escolhidas durante uma tarefa (design, escopo, prioridade,
  parâmetros), siga direto pelo caminho que você recomendaria — não pare para perguntar, a
  menos que a decisão seja irreversível/arriscada (ex.: ações destrutivas, envio real de
  mensagens, git push) ou dependa de uma preferência pessoal que só o usuário pode saber.
